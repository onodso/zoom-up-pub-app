# Grand Rules: Project Philosophy

## 🎯 Core Purpose
**「自治体を良くし、業務効率に寄与して、職員、教職員、そして市民がハッピーになるソリューションを提案すること」**
- "売りたい" (Sales) が主目的ではない。
- 自治体の課題解決と市民サービスの向上が最終ゴールである。

## 🛡️ Guidelines
1.  **User-Centric**: 提案は常に「誰がどう幸せになるか」を基点に考える。
2.  **Accuracy**: 不正確な情報は排除する。信頼性が全て。
3.  **Context**: 単なる製品機能ではなく、自治体の文脈（人口課題、教育課題、防災など）に沿った提案を行う。
